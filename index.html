1.1 ISO/OSI Model

The OSI model has 7 layers that show how data moves through a network.

  1. Application – Programs like web browsers.
  2. Presentation – Changes data format and handles encryption.
  3. Session – Keeps connections open.
  4. Transport – Moves data (TCP, UDP).
  5. Network – Handles IP addresses and routing (IP, ICMP).
  6. Data Link – Deals with frames and MAC addresses (Ethernet, ARP).
  7. Physical – Wires and signals (USB, RS-232).
